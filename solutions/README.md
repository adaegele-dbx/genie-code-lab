# Solutions — Reference Material

Everything in this folder is a **fallback** for when Genie Code's output
during the lab doesn't quite work, or for post-lab study. The intended flow
is that attendees prompt Genie Code in their own words at each step; these
files are only used when something gets stuck.

| File | Used in | Purpose |
|------|---------|---------|
| `sample_pipeline.py` | Part 3a | Known-good Lakeflow SDP if Genie Code's generated pipeline won't run |
| `sample_genie_instructions.md` | Part 3b | Description + tuning text for the Genie space |
| `sample_app/` | Part 3c | Known-good Streamlit app (`app.yaml`, `app.py`, `requirements.txt`) |
| `sample_assistant_instructions.md` | Part 5 | The instructions block to paste into `.assistant_instructions.md` |
| `sample_skill/SKILL.md` | Part 6 | The reference skill to drop into `.assistant/skills/anomaly-triage/` |
| `reference_prompts.md` | All parts | Every suggested prompt collected in one place for fast copy-paste |
